# Next Lift

Personal, local-first fitness log app for iOS and Android using Flutter.

## Current status

- Product and architecture specs are in `docs/specs/`.
- Theme/design token system is scaffolded under `lib/core/design_system/`.
- App shell and Riverpod wiring are present in `lib/`.

## Local setup

1. Install Flutter SDK and add it to your shell `PATH`.
2. From repo root, run:
   - `flutter pub get`
   - `flutter doctor`
3. Launch app:
   - `flutter run`

## Key default

- Default theme: `graphite_teal`.
